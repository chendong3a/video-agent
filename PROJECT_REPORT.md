## 一、项目概述

### 1.1 项目定位

**视频智能体**是一个端到端的 AI 驱动视频内容再创作平台。用户只需提供一个视频链接（B站/抖音），系统即可自动完成内容分析、文案改写、声音克隆、口型对齐、字幕合成等全流程，最终输出一个全新的口播视频。

### 1.2 核心价值

| 痛点 | 解决方案 |
|------|----------|
| 视频创作门槛高 | AI 自动化全流程，零基础也能创作 |
| 内容改写耗时 | 4种风格一键生成，秒级出稿 |
| 配音成本高 | 声音克隆技术，上传样本即可复制 |
| 口型不匹配 | AI 口型对齐，效果自然逼真 |
| 后期制作复杂 | 字幕+标题+BGM 自动合成 |

### 1.3 目标用户

- 短视频创作者 / 自媒体博主
- 知识付费内容生产者
- 电商带货视频制作团队
- 企业宣传视频制作人员

## 二、项目功能

### 2.1 核心功能模块

```
┌─────────────────────────────────────────────────────┐
│                    视频智能体                         │
├─────────┬──────────┬──────────┬──────────┬──────────┤
│ 视频分析 │ 文案生成  │ 声音克隆  │ 口型对齐  │ 后期合成  │
│ BiliNote │ MiniMax  │CosyVoice3│ MuseTalk │  FFmpeg  │
└─────────┴──────────┴──────────┴──────────┴──────────┘
```

#### 模块 1: 智能视频分析 (BiliNote)
- **输入**: B站/抖音视频链接
- **输出**: 结构化内容 (标题、转录文本、摘要、要点、章节)
- **能力**: 自动识别平台、提取视频ID、调用 BiliNote API 分析
- **状态**: ✅ 模块代码完成

#### 模块 2: 多风格文案生成 (MiniMax API)
- **输入**: 视频分析结果
- **输出**: 口播文案 (含字数、预估朗读时长)
- **4种风格**:
  - 🎓 **专业风格** - 严谨有深度，信息密度高
  - 😄 **幽默风格** - 轻松有趣，有梗有料
  - 💖 **情感风格** - 走心共鸣，触动人心
  - 🔥 **带货风格** - 激发购买，强力转化
- **状态**: ✅ 模块代码完成 (含 Mock 数据降级)

#### 模块 3: 声音克隆 (CosyVoice3)
- **输入**: 声音样本 (几秒即可) + 文案文本
- **输出**: 克隆声音的音频文件
- **能力**: 零样本声音克隆、长文本 TTS、音频后处理
- **状态**: ⏳ 框架已搭建，待实现

#### 模块 4: 口型对齐 (MuseTalk)
- **输入**: 底片视频 + 克隆音频
- **输出**: 口型同步的视频
- **能力**: 人脸检测、口型生成、视频合成
- **状态**: ⏳ 框架已搭建，待实现

#### 模块 5: 后期合成 (FFmpeg + MoviePy)
- **输入**: 口型视频 + 音频 + 文案
- **输出**: 最终成片 (含字幕、标题、BGM)
- **能力**: 
  - faster-whisper 自动生成字幕 (SRT)
  - FFmpeg 字幕烧录 + 自定义样式
  - BGM 混音 (淡入淡出、音量平衡)
  - 标题叠加
- **状态**: ⏳ 框架已搭建，待实现

### 2.2 用户交互流程

```
用户操作                          系统处理
────────                          ────────
1. 粘贴视频链接          →    BiliNote 分析视频内容
2. 选择文案风格          →    MiniMax 生成口播文案
3. 上传声音样本          →    CosyVoice3 克隆声音
4. 上传底片视频          →    MuseTalk 口型对齐
5. 点击生成              →    FFmpeg 后期合成
6. 预览/下载成片         ←    输出最终视频
```

## 五、发展期望

### 5.1 短期目标 (8天冲刺内)

1. **MVP 可用** - 完整跑通"链接→成片"全流程
2. **效果达标** - 声音克隆自然度 > 80%，口型同步率 > 85%
3. **性能可接受** - 单个视频处理时间 < 10分钟
4. **PWA 可安装** - 移动端体验良好

### 5.2 中期目标 (1-3个月)

1. **多平台支持** - 扩展到 YouTube、小红书、快手等平台
2. **批量处理** - 支持批量视频再创作
3. **模板系统** - 预设视频模板（开箱、测评、教程等）
4. **用户系统** - 注册登录、历史记录、收藏夹
5. **云端部署** - 支持多用户并发，GPU 资源调度
6. **质量优化** - 声音克隆和口型对齐效果持续提升

### 5.3 长期愿景 (3-12个月)

1. **AI 编导** - 自动选择最佳素材、剪辑节奏、转场效果
2. **多语言** - 支持中英日韩等多语言视频生成
3. **实时预览** - 边编辑边预览，所见即所得
4. **API 开放** - 提供 SaaS API，供第三方集成
5. **数字人** - 集成数字人形象，无需真人底片
6. **商业化** - 
   - 免费版：每月 5 个视频
   - 专业版：无限视频 + 高级功能
   - 企业版：私有部署 + 定制开发
